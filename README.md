# AMA ai — Premium Prompt Vault 🚀

Welcome to the official prompt database for the AMA ai extension. 

As a visual creator, I got tired of generic, boring AI generations. This vault is specifically engineered for professional visual production. You will not find basic close-up portraits or flat backgrounds here. These prompts are designed for comprehensive, creative camera angles, complex artistic compositions, and cinematic lighting.

## 📥 How to use this file:

1. Download the free AMA ai extension from the Chrome Web Store: [https://chromewebstore.google.com/detail/iogcommbfgblfcjncaknikepgcbilffe?utm_source=item-share-cb]
2. Download the `ama_prompts.json` file from this repository.
3. Open the AMA ai extension settings and click **Import Database**.
4. Select the JSON file. 

All premium prompts are now instantly available in your local browser storage, ready to be injected into ChatGPT, Claude, or routed directly to your n8n workflows via the Spotlight search overlay (Ctrl+Shift+K).
